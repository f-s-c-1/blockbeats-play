# 燃团 · 主持人手册（台词稿 + 控制台操作对照）

> 用法：每个环节分两栏——**【你说】**是可以直接照念的台词，**【你做】**是控制台上对应要点的按钮。
> 控制台地址：`http://107.173.49.32:53917/admin`
> 铁律：游戏在线下玩，屏幕只负责发牌、计时、记分、开奖。说完台词再按按钮，节奏永远在你手里。

---

## 〇、赛前准备（活动前一天）

**【你做】**
1. 打开控制台 → 创建房间：房间码设一个好念的（如 `2024`），**主持口令一定要设**（换设备/手机没电换机时凭它找回后台）；
2. 把"复制链接"发到活动群，让大家提前点开加入、起名、选头像；
3. 人到齐后在「抽签」标签**提前随机分队**：选队数 → 「随机分队」→ 不满意就重抽到满意为止（**只要不点"揭晓分组"，玩家就什么都看不到**）；
4. 在「成员与内鬼」里**提前勾选 2-3 个通过气、玩得开的人为内鬼**，点「指定内鬼」（同样不会下发，揭晓分组那一刻才通知到本人）；
5. 翻一遍「游戏手册」标签，把今天要玩的项目过一遍；
6. 准备一支笔一张纸，关键积分手写备份一份（系统有记分流水，但双保险不吃亏）。

## 一、开场白（5 分钟）

**【你说】**

> 各位！欢迎来到今天的团建现场！😄
>
> 在开始之前，先说清楚今天最重要的一个规则——**把手机掏出来！** 对，你没听错，今天是我唯一一个鼓励你们玩手机的人。
>
> 大家扫群里的链接，或者打开浏览器输入屏幕上这个地址，进入房间之后**起个名字、挑个头像**。注意：这个名字会跟着你一整天，会上积分榜，会被全场投票，慎重一点，但也别太正经。
>
> 第二件事，把今天的玩法说清楚——**今天不是各玩各的，是团队战！** 待会儿每个人都会有自己的队伍。从现在起到晚上，所有游戏——不管是动嘴的、动手的还是斗智的——**全部按队伍算总分**：你抢答赢一题，是你们队加分；你被淘汰，是你们队丢人；今晚结束时积分最高的队，拿走冠军和奖品。所以一句话：**你的每一次表现，都在为你的队打工。**
>
> 进去之后你会看到"等待主持人"——对，今天你们的手机就是我的遥控器。我不推内容，你们的屏幕就是一块砖；我一推，每个人看到的东西**可能都不一样**——有人会拿到只有自己能看到的词，有人会收到秘密任务。所以记住第三条规则：**你的屏幕只能你自己看。**
>
> 都进来了吗？我这边看到已经有 XX 人了，还差几位，旁边的互相催一下。

**【你做】**
- 「成员」标签实时看"已加入 X/X"，等齐人；
- 有人起名不当 → 点他名字旁的「改」直接改掉；陌生人乱入 → 「×」踢出。

## 二、分组揭晓（10 分钟）

> ⚠️ 适用前提：你已经**提前在后台分好了队、塞好了内鬼**（赛前准备第 3 步）。现场只做"揭晓"这一个动作——所以台词不说"现在随机分"，说"队伍已经定了"，制造开盲盒的感觉。

**【你说】**

> 刚才说了今天是团队战——那你的队友是谁？
>
> 告诉大家：**队伍已经分好了**，就在你们的手机里。是兄弟还是冤家，是大腿还是队友，我数三二一，所有人同时看手机开盲盒——
>
> 三——二——一——**看！**

**【你做】**
- 点「**揭晓分组**」（就这一下，分组和内鬼身份同时下发到每个人手机）。

**【你说】**（揭晓后）

> 找到组织了吗？！现在给大家 3 分钟，做三件事：
>
> 第一，**找到你的队友站到一起**——对，物理上站到一起，今天一整天你们就是一个战壕的；
>
> 第二，选队长！系统随机指了一个**临时队长**（屏幕上有 👑 的那位，举手让我看看！）——但谁真正带队，你们自己商量。想换人？临时队长在自己手机的队伍面板里，点新队长名字旁边的「**移交**」就行，👑 当场易主。队长手里是有实权的：改队名、待会儿大项目里掷骰子做决策，都是他说了算——选个靠谱的，别选最帅的；
>
> 选好了队长就改队名，给你们队起个响亮的名字，它会挂在积分榜上一整天，起得土，输了都没人同情你们；
>
> 第三，也是最重要的——**有些人刚才的手机上，比别人多了一块内容。**（停顿，扫视全场）我不说是什么，收到的人自己心里清楚。管好你的屏幕，管好你的表情，你的任务贯穿今天一整天，晚上会有专门的环节算总账。其他人也别凑过去看，**从现在起，每个人都有嫌疑。**

**【你做】**
- 等各队站位、选队长、改队名（积分榜自动同步）；
- 哪队的队长不肯交权/手机没电选不了 → 「抽签」标签队伍卡里点「设队长」直接帮他们定；
- 内鬼身份已随揭晓自动下发到本人（含任务卡），全程只有你和他本人知道；
- 想中途给内鬼加戏 → 「成员」标签他名字旁的「派任务」，悄悄改派新任务（他手机只震动提醒，旁人无感）。

> 💡 小组对抗的氛围靠你反复"点火"：每个游戏开始前加一句**"这是团队战，赢了 XX 队加 X 分"**；每轮结束推一次积分榜让落后的队着急。台词模板："目前 XX 队领先！XX 队，你们就甘心吗？"

## 三、热场：疯狂故事组合（10 分钟）

**【你说】**

> 第一个游戏，热热身，动手不动腿——**疯狂故事组合**！
>
> 每个人手机上现在有三个框：**一个人名、一个地点、一件正在做的事**。人名建议写在场的人，地点和事情越离谱越好。放心大胆写，系统会把你们所有人的素材**打乱重拼**，没人知道哪句是谁写的。
>
> 都提交了吗？好——见证奇迹的时刻！

**【你做】**
- 「故事组合」标签 → 「开始收集」→ 盯着"已收 X 份"催交 → 点「🎲 抽一条」开奖（可以连抽三五条，笑点足的那条给素材提供者的队加分）。

## 四、谁是卧底（20 分钟）

**【你说】**

> 经典环节——**谁是卧底**！规则九成人都会，给新同学说一遍：
>
> 每人手机上发一个词，大多数人的词一样，**有人不一样**，还可能有人**根本没有词**（白板，全靠演）。每人轮流用一句话描述自己的词——**不能带词里的字**。一轮描述完投票，票最多的出局亮牌。平民找出卧底就赢，卧底活到最后卧底赢。
>
> 注意你们的手机现在是**防偷看模式**：词要**按住才显示，松手就盖**，所有人的卡背一模一样，所以谁也别想瞄出什么。
>
> 第一轮，从 XXX 开始描述！

**【你做】**
- 「卧底」标签：选词对分类（或勾"自定义词对"现编一对内部梗）→ 设卧底数/白板数 → 勾选参与者 → 「发词」；
- 投票揪人：点「本局投票」→ 大家投完点「公布票数」；
- 出局的人：在发牌明细里点他的「出局」，他手机会显示已出局；
- 结束后给获胜方记分（积分标签或快捷键）。

## 五、你比我猜 / 猜猜我是谁（各 15 分钟）

### 你比我猜

**【你说】**

> 下一个——**你比我猜**！每轮一位比划者，词**只发到他一个人的手机上**，可以动可以叫，就是不能说词里的字。其他人看大屏倒计时，60 秒，猜中越多分越高！

**【你做】**
- 「你比我猜」标签：选比划者 + 选词分类 → 「发词给比划者」（倒计时自动起跑）；猜中一个换一个词，结束记分。

### 猜猜我是谁

**【你说】**

> 这个游戏反过来——**你头上有个词，全场都看得见，就你自己看不见**。轮到你的时候提问，大家只能回答"是"或"不是"——"我是活的吗？""我能吃吗？"猜中自己头上的词就赢。
>
> 没被选中的同学也看手机，你们是**全知视角**，憋住，谁剧透谁表演节目。

**【你做】**
- 「猜猜我是谁」标签：选分类 → 勾参赛者（建议 4-6 人）→ 「发牌开始」；谁猜中点谁的「猜中」，他手机当场翻牌庆祝。

## 番外：大富翁（团队对抗，约 30 分钟，可插在任意时段）

**【你说】**

> 重头戏来了——**真人大富翁**！你们每个队就是棋盘上的一颗棋子，开局每队 25 金币，24 格棋盘上有 12 块地等着抢。骰子只在**队长**手机上——开局前最后 30 秒，确认你们的队长是谁！买地、用道具、保不保释全是他拍板，押错人这局就别怪系统。想换，现在让队长点「移交」还来得及。
>
> 丑话说在前：**欠债会被系统强制变卖地产抵债，卖光还不上就破产出局**，只能看别人玩——别梭哈到兜里一个子儿不剩。
>
> 轮到你们队，队长就把骰子拍下去！掷出**对子**还能再来一次——但连掷三对，直接进拘留所！
>
> 不是队长也别闲着：每回合开骰前，**所有人都可以在手机上押点数**，猜中两颗骰子的和，你的队白拿 1 金币！
>
> 重点说说赚钱：买地收租，**同色的两块地凑成一套，租金翻倍**！机会卡里还能捡到三件神器——🎮 遥控骰子想走几步走几步，🚧 路障埋在路上坑人，🛡️ 免租卡保你免单一次。踩到 😈 惩罚格，全队推一个人当场执行大屏上的惩罚！

**【你做】**
- 前置：必须先「抽签分组」并**揭晓分组**（队长才能在自己手机上掷骰）；
- 「大富翁」标签 → 「开局」：全场手机出现棋盘，第一队开始；
- 正常情况你不用动手：队长掷骰/买地/用道具/保释全自助，回合自动轮转，棋子逐格走动画全场同步；
- 队长手机没电/卡住 → 你随时「🎲 代掷骰子」「代买/不买」「💰 代保释」「⏭️ 跳过回合」兜底；
- 想换人当队长 → 队长在自己手机的队伍面板点队友旁的「移交」，或你在「抽签」标签的队伍卡里点「设队长」强制改任，掷骰权立刻跟着走；
- 有队踩中 😈 惩罚格 → 大屏直接显示惩罚内容，盯着他们执行完再继续；
- 嘴瓢忘了刚才发生什么 → 面板底部「事件回放」存着最近 8 条；
- 控制时长：转 3-4 圈（看面板"第 X 圈"）就可以「结算」→ 总资产排名上屏 → 「冠军队 +3」记入活动积分；
- 不点结算也有保险：只剩一队存活、或跑满 10 圈，系统自动结算。

**【节奏提示】**
- 金币和活动积分是两套账——大富翁里破产不影响总积分，放心让他们梭哈；
- 拘留所有两条路：花 2 金币保释当回合就能掷，或认栽跳过——逼队长当众做选择，喊一句"队长，队友看着你呢"；
- 谁家凑成套了（地产总览有 🔗 标记），及时帮全场拉仇恨："注意！小吃街已经是 XX 队的天下了，踩进去交双倍！"

>
> 投票开始，每人一票，投给你心中最可疑的人。内鬼也要投，建议投个无辜的演一下。
>

**【你做】**

> 加场玩法：**真真假假**——投票面板用"选项投票"，让嫌疑人说三件事（两真一假），全场投哪件是假的。

## 附：应急预案

| 状况 | 处理 |
|---|---|
| 你的手机断线/没电 | 任何设备打开控制台 → 「找回已有房间」→ 房间码 + 主持口令，秒回现场 |
| 玩家说"我没收到" | 让他回到前台/刷新——系统会自动恢复到当前画面，身份积分不丢 |
| 现场网络很差 | 屏幕环节先跳过，玩纸笔能玩的（手册里口头类全是零道具），分数你纸上记，网好了补录 |
| 有人中途加入 | 发链接直接进，手动在抽签前/后把他拖进队（重新分队会打乱已有分组，慎用） |
| 误操作记分 | 「积分」→ 「↩ 撤销上一笔」，流水里每笔都有时间可对账 |
| 大屏内容乱了 | 顶栏「全体等待」一键拉回，重新推 |

> 最后一句给主持人自己：**系统是道具，你才是节目。** 台词稿是底稿，现场抖出来的包袱永远比稿子好笑。
